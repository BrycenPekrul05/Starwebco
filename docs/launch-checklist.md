# Website Launch Checklist

## Before Launch
- Review all pages on desktop
- Review all pages on mobile
- Test all buttons and links
- Test contact form
- Check spelling and grammar
- Add page titles
- Add meta descriptions
- Add image alt text
- Compress images
- Confirm domain settings
- Confirm SSL / HTTPS

## After Launch
- Submit site to Google Search Console
- Set up analytics
- Confirm contact form delivery
- Send client final link
- Ask for testimonial
- Offer maintenance plan
